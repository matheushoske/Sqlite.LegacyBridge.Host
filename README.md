# Sqlite.LegacyBridge.Host

Binary-focused repository for the .NET Framework host that opens legacy encrypted SQLite and serves named-pipe bridge requests.

## Contains

- `src/Sqlite.LegacyBridge.Host`
- `src/Sqlite.LegacyBridge.Protocol`

## Release

- GitHub Actions workflow: `.github/workflows/host-release-sqlite-legacy-bridge.yml`
- Releases are created from tags starting with `host-v`.
